# Mi Amor Vocabulary Quiz ❤️

A vocabulary learning website built with HTML, CSS, and JavaScript.

This project was created as a personalized study guide and includes multiple game modes to make learning new vocabulary more fun and interactive.

## Features

- Full Vocabulary Quiz
- Quick Quiz (10 random words)
- Practice Wrong Words Mode
- Timed Vocabulary Test
- Quiz Block Puzzle Game
- Picture Puzzle Game
- Randomized Questions and Answers
- Vocabulary Loaded from JSON
- Mobile-Friendly Design
- Score Tracking

## Technologies Used

- HTML
- CSS
- JavaScript
- JSON

## How It Works

Vocabulary words are stored in a JSON file with:

- Word
- Explanation
- Example Sentence
- Fill-in-the-Blank Example
- Correct Answer

The website loads the vocabulary file and generates quizzes automatically.
